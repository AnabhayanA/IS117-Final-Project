## Development Setup

### Install Dependencies
```bash

```

### Linting
```bash

npm run lint:html     # HTML only
npm run lint:css      # CSS only
```

### Local Development
Use Live Server extension or:
```bash
python3 -m http.server 8000 --directory docs
```
